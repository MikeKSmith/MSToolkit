Status

The MSToolkit v2.0 was release to the public via R-Forge on 29th June 2009.  MSToolkit was designed by Mike K Smith of Pfizer and was developed by Mango Solutions for Pfizer. The first release of this software was an internal release to Pfizer. The current version (build) is 2.0. MSToolkit is available via CRAN. MSToolkit source code, news, bug and feature requests and forums are available through R-Forge. Production releases of MSToolkit will be submitted to CRAN. Development of MSToolkit with bug fixes, new features and additional functionality will be made available through the daily build cycle at R-Forge. For stable releases, please use the CRAN version.
